{% page_help_embedded title='VUS Resolution' %} You can review which of your classifications have overlaps with other labs where all records are VUS.

There are columns provided for the following:
  • The maximum absolute ACMG point score for this allele. Some labs use curation methods that don't map properly to ACMG point system.
  • Number of patients for the given Allele - typically measured by number of classifications. Some labs are unable to provide patient counts.
  • Hover over the headings to see the descriptions. The header will have one of the following icons
    • Indicates all classifications share the same strength for this criteria.
    • Indicates all classifications have a strength for this criteria, but one or more is "unspecified". Could indicate a difference.
    • Indicates that classifications for this criteria have different strengths.
