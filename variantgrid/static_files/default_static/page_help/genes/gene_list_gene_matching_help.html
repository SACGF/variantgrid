GeneList records are processed as follows:

  • original name is matched to a known symbol (perhaps via an alias which will be set in the alias column
  • For each GeneAnnotationRelease - map the matched symbols to a Gene ID (or IDs)

In an analysis GeneListNode - filters variants where any transcript matches the Gene ID

Warning: If a row has no Gene ID for a genome build, it will not filter any variants in an analysis using that build.
