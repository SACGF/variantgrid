Variant Annotation Runs track the pipelines on the server to annotate variants.

VariantGrid stores each variant once, but multiple versions of annotation per variant. See annotation version. for these versions.

AnnotationRuns are created automatically whenever VCFs with previously unknown variants are uploaded.

The annotation run dumps new variants to the server, runs them through the pipeline and re-uploads the data.

The heavy VEP step can also be run off the server (e.g. on a separate compute box) and the annotated results re-imported. See External Annotation.
