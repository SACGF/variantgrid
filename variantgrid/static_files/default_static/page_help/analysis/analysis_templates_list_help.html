When you click the "Save version" on the analysis template page, you create a new version of the analysis template.

The versions are listed on this page. Only the latest ("active") version can be used. If you have write permissions you can change the following fields on the latest / active version to show/hide analysis templates in various places.

Name template

Used to create the name of a new analysis made from the template. This is a Python 'format string' template, so eg %(input) is replaced with the name of the sample/cohort/pedigree used.

Autocomplete

Does the template appear in the 'Analysis template' dropdown box in the "Create Analysis" section at the bottom of sample etc pages.

Links

Does the template appear as a clickable link at the bottom of sample etc pages.

Requires Somatic

Template will only appear in autocomplete/link if sample is of 'Variants type' of Mixed/Somatic only

Requires Sample Gene List

Template will only appear in autocomplete/link if sample has a sample gene list
