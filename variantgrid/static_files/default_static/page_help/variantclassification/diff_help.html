This page shows the difference between multiple variant classifications for the same variant, or differences between multiple published versions of the same classification.

By clicking on "Configure columns" you can hide some classification columns from the view.

For any given row, there will be an icon to indicate if there were differences.

  • If there are at least two different non-blank values, the row will have a red difference icon.
  • If all non-blank values are the same, the row will have a grey tick.
  • If all values are the same and there are no blanks, the row will have a green tick.
Hover over the row label for details about the differences and a description of the evidence key.
