Missing annotation

Structural Variants are missing some normal annotation:

  • Sequence and domains
  • Pathogenicity
  • Conservation
  • Splicing
  • Non-gnomAD population frequency
  • COSMIC
  • MAVEdb
We hope to be able to add some of this in the future.

gnomAD

We report overlaps against gnomAD Structural Variants that:
  • Overlap >= 80% with our SV
  • Have no FILTERS (there were too many extremely large artifacts that overlapped with everything)
  • Have the same "type" (ie del for a del, inv for an inv)

You can view all overlaps by selecting "Show Structural Variant Overlaps"

If there are multiple overlapping gnomAD records, we assign the one with the lowest allele frequency to the gnomAD values (gnomad_af, allele count) etc - it is this which will be used for filtering in an analysis.
