Be aware of transcript choice. Usually the most deleterious transcript is chosen for curation, however, it is also important to investigate relevant tissue-specific expression and understand is the canonical transcript and whether this is the same transcript that has been previously used for curation.

  • Representative - transcript chosen for this variant (via VEP pick) - this is what is used in an analysis and shown in the grid.
  • Canonical - transcript for gene, chosen by annotation provider (RefSeq/Ensembl)

The ClinGen Allele Registry is useful to find transcripts.
