Configure the parameters here to generate a re-usable export URL that can be periodically called to download shared classifications.
If you wish to download a few classifications for personal reviewing, you may be better served by the download links on the classifications page.

If you are going to copy the URL in your browser to download periodically, choose "Web Browser".
If this is for copying into some a program that will automatically download, choose "API".

The following classifications will not be downloaded (except when using the CSV format)

  • Classifications that are matching or failed to match a variant
  • Classifications with outstanding matching warning or transcript version flags
  • Classifications attached to variants that have liftover warning flags (if the liftover warning is for the same transcript as the classification)
  • Classifications that could not be lifted over to the desired genome build

Pick a genome build, regardless of which build the classification was uploaded as it will be lifted over to your desired build.

You can choose to exclude entire organisations, or only certain labs within an organisation. If you are uploading classifications to this environment and only wish to download other labs classifications, these options will let you do that.

Lastly, choose a file format that your system can easily handle
Some formats will contain only a summary of data e.g. MVL and VCF
Some formats will have one record per classification e.g. CSV, JSON while others might group on variant/transcript e.g. MVL
