This page will list all classifications belonging to your lab/s that have flags raised against them.
Each classification flag indicates that there is an action that needs to be performed against the classification.
Clicking on each flag will give you instructions on what needs to be done to resolve it.

Flags range from issues with variant matching, to discordances or suggestions.
You can filter by the flag categories below:

Have any issue: All classifications with flags requiring action will be shown.

Requires discussion: Shows classifications in discordance, where a user has raised a suggestion or if a clinical significance change has occurred.

Requires a data fix: The classification is missing mandatory fields, has not been shared with Shariant users, has unshared edits or has an outstanding warning/error regarding variant matching.
This will most likely require a technical representative from your lab to fix.

Have been withdrawn: Here you can view classifications from your lab that have been withdrawn. No action is required here.
