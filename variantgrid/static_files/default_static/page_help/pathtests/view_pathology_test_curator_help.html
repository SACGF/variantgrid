You CANNOT rename a pathology test - the name is a fundamental part of a test and we need to be able to find any historical orders. Instead of renaming a test, create a new test by copying genes from the old one, then delete the old test. Steps:

  • Click [manage tests] on the left menu
  • Click [new pathology test]
  • Enter the new name, select the old test to copy from then click [create].
This will create a new test based on the old one. You can then delete the old test, by going to the test page and clicking the [Delete...] link.
