Click data in the left menu to see samples from existing VCFs, or upload to add your own VCF. Feel free to right click and open multiple tabs!

This will be processed then once done you'll get a message (top right) and it will appear on the data page.

Perhaps while waiting you could search for an existing variant such as: "NM_001080463.1:c.5972T>A".

After that, perhaps try an analysis (top menu), search for your favorite gene or open a public sample.
