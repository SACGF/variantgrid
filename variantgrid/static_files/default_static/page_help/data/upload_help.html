File types are matched according to their file extension as per below:
Extensions Details
Genotype VCF .vcf See specification at 4.1 format. Allele Depth is read from "AD" then "DP" INFO fields, Phred likelihoods from 'PL' then 'GQ'. This is standard GATK (AD/PL), and Torrent Formats (DP/GQ)
Bed File .bed, .broadPeak, .narrowPeak bed spec, broadPeak spec, narrowPeak spec. Either "chr1" or "1" chromosomes OK.
Gene List .tsv, .csv, .txt This can be either Gene Symbols (GAPDH) or Ensembl Transcript IDs (start with "ENST")
Gene Coverage .txt Must have columns: 'Gene/Exon', 'Min Coverage', 'Mean Coverage', 'Standard Deviation Coverage', '% bases >20x', '% bases <10x', '% bases @ 0x', 'Estimated Sensitivity of Detection'
Patient Records .csv See import patient records details. Sample Patient Upload CSV .
Ped .ped PED/Linkage file format.

GZipped files are supported: They must end in .gz, which will be removed and the extension matched as above.
