VCF sample name

The name from the VCF, this cannot be changed. Sample name is initially set to this but can be changed.

BAM file path

The path to a BAM file, used with IGV integration (Settings -> IGV Integration) to automatically load bams with IGV links.

Variants Type

Variants type can be 1 of 4 values:

  • Unknown
  • Germline
  • Mixed (single Sample)
  • Somatic only (Tumor minus normal)

When a VCF is imported, we look at the "source" field in the header. If it contains "freeBayes" it is set to "Mixed". If it contains "combine_caller_data" (tumor-normal pipeline) then it's set to "Somatic only". Otherwise it's left as "unknown"

Mutational signatures are calculated for Somatic only samples, and some analysis templates only appear for certain variant types.
