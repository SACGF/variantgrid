GenomicIntervals file comes from uploading a Bed File, which is flattened and merged (overlapping intervals are combined) using BedTools.

Genome Build

We attempt to automatically detect the genome build via:
  • track header line, contains a recognised build name or alias in the "db", "genome", "genome_build" or "genomeBuild" fields
  • detect from filename (contains ONLY ONE recognised build name or alias in the filename, eg "intervals_grch37.bed")
  • If the user only has 1 genome build in their User Settings
