Pathogenicity Thresholds

VariantGrid filters and colours missense pathogenicity prediction scores using the ClinGen-calibrated PP3 (pathogenic supporting) and BP4 (benign supporting) raw-score cutoffs below. A score ≥ the PP3 threshold supports pathogenic; a score ≤ the BP4 threshold supports benign; values in between are indeterminate. Tools without a published ClinGen calibration use the author-recommended cutoff (single threshold) or, where noted, have no calibrated threshold.

These thresholds drive the colour bands on the analysis Effect (Damage) node sliders and the Pathogenicity card on the variant details page.
